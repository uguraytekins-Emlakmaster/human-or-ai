## Gizlilik politikası URL

- Uygulama içinde kullanılan link: `src/constants/appLinks.ts` içindeki `PRIVACY_POLICY_URL`.
- Hazır sayfa: `docs/privacy.html`. GitHub Pages’te repo ayarlarından Source = main, klasör = /docs seçin; yayın sonrası adres `https://KULLANICI.github.io/REPO_ADI/privacy.html` olur — bu adresi `appLinks.ts`’e yazın. Metin kaynağı: `docs/PRIVACY.md`.

## App Store ID (iOS)

- iOS’ta “Uygulamayı değerlendir” butonu için: App Store Connect’te uygulama yayınlandıktan sonra aldığınız sayısal ID’yi `src/constants/appLinks.ts` içindeki `APP_STORE_ID` alanına yazın.
