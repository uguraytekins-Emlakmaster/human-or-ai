# App assets

Mağaza görselleri:

| Dosya | Boyut | Açıklama |
|-------|--------|----------|
| `icon.png` | 1024×1024 | App Store / Play Store ikonu |
| `adaptive-icon.png` | 1024×1024 | Android adaptive icon |
| `splash.png` | 1284×2778 | Açılış ekranı |

Yeniden üretmek için:

```bash
npm run generate-assets
```

Script: `scripts/generate-assets.py` (Pillow gerekir).
